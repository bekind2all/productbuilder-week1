# 프로젝트 청사진: 로또 번호 추첨기

## 개요

이 프로젝트는 사용자가 버튼 클릭 한 번으로 무작위 로또 번호를 생성할 수 있는 간단한 웹 애플리케이션입니다. 이 애플리케이션은 순수 HTML, CSS, JavaScript를 사용하여 구축되며, 재사용 가능한 UI 요소를 위해 웹 컴포넌트를 활용합니다.

## 기능 및 디자인

### 핵심 기능

*   **번호 생성:** 사용자는 "번호 생성" 버튼을 클릭하여 1부터 45까지의 숫자 중 6개의 고유한 무작위 번호를 생성할 수 있습니다.
*   **결과 표시:** 생성된 번호는 시각적으로 구분되는 공 모양의 요소 안에 표시됩니다.

### 디자인 및 스타일

*   **레이아웃:** 중앙 정렬된 콘텐츠와 함께 깔끔하고 현대적인 디자인을 채택합니다.
*   **색상 팔레트:** 다채로운 색상을 사용하여 각 번호 공에 고유한 색을 부여하여 시각적 매력을 더합니다.
*   **타이포그래피:** 가독성 높은 글꼴을 사용하여 제목과 번호를 명확하게 표시합니다.
*   **애니메이션:** 번호가 생성될 때 부드러운 애니메이션 효과를 추가하여 사용자 경험을 향상시킵니다.
*   **반응형 디자인:** 다양한 화면 크기에서 잘 보이도록 반응형 레이아웃을 적용합니다.

## 기술 스택

*   **HTML:** 애플리케이션의 기본 구조를 정의합니다.
*   **CSS:** 시각적 스타일링과 레이아웃을 담당합니다.
*   **JavaScript:** 로또 번호 생성 로직과 웹 컴포넌트를 구현합니다.
*   **웹 컴포넌트:** 재사용 가능한 `<lotto-generator>` 커스텀 엘리먼트를 만들어 애플리케이션의 핵심 기능을 캡슐화합니다.

## 현재 요청 계획

1.  **`index.html` 수정:**
    *   페이지 제목을 "로또 번호 추첨기"로 설정합니다.
    *   스타일시트(`style.css`)와 자바스크립트 파일(`main.js`)을 연결합니다.
    *   `<lotto-generator>` 커스텀 엘리먼트를 추가하여 애플리케이션의 본문을 구성합니다.

2.  **`style.css` 수정:**
    *   전체 페이지에 대한 기본 스타일(배경색, 글꼴 등)을 설정합니다.
    *   로또 번호 공의 스타일(크기, 색상, 애니메이션 등)을 정의합니다.
    *   "번호 생성" 버튼의 스타일을 지정합니다.
    *   미디어 쿼리를 사용하여 모바일 장치에서도 보기 좋게 만듭니다.

3.  **`main.js` 수정:**
    *   `LottoGenerator`라는 이름의 웹 컴포넌트 클래스를 정의합니다.
    *   컴포넌트 내에서 1부터 45까지의 숫자 중 6개의 고유한 무작위 번호를 선택하는 로직을 구현합니다.
    *   생성된 번호를 화면에 렌더링하는 함수를 만듭니다.
    *   "번호 생성" 버튼 클릭 시 번호 생성 및 렌더링 함수를 호출하는 이벤트 리스너를 추가합니다.
    *   `customElements.define`을 사용하여 `<lotto-generator>` 태그를 등록합니다.
